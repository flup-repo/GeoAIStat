# GeoAIStat

GeoAIStat is a static-first React app for exploring AI adoption geography across countries and US states. The current slice implements the single-route experience, sample processed data artifacts, URL-driven state, and a cinematic globe shell.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test:run`

## Data

- Source artifacts are generated into `public/data/` by `npm run build:data`.
- The current provider adapters are seeded with representative sample values so the app can be verified locally before real ingestion work is connected.
